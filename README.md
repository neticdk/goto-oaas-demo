# GOTO Aarhus 2021 - OaaS Demo

The respository contains source code for Netic talk at GOTO Aarhus 2021. The repository
is organized as follows.

- `gotk` - manifests reconciled by GitOps Toolkit
- `manifests` - manifests that are manually applied to the cluster during the demonstration
- `terraform` - Terraform code to provision AKS and EKS demonstration clusters
